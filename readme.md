# cesar-ebac.frontend-starter

## Objetivo del proyecto
Proyecto frontend de lista de tareas (To-Do List) desarrollado como práctica del Módulo 3 del Bootcamp de IT para principiantes de EBAC. El objetivo es aplicar conocimientos de HTML, CSS y estructura de proyectos web con control de versiones en GitHub.

## Páginas disponibles
- index.html — Vista principal con lista de tareas
- about.html — Página de información sobre el proyecto y el autor
- nueva-tarea.html — Vista para agregar nuevas tareas
- tareas-mensuales.html — Vista de tareas mensuales
